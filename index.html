Send photos/videos to this bot to add them to your event gallery

Available commands:

  • /start - Create a new event
  • /done - Finish uploading and get your link
  • /help - Show help information

How It Works

  1. Message @EventMediaBot on Telegram with /start
  2. Send photos and videos directly to the bot
  3. When finished, send /done to receive your event link
  4. Create a QR code from the link and share with guests

All media is stored on Telegram's servers - no storage costs for you!
